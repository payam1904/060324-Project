Introduction to HTML

HTML stands for Hyper Text Markup Language. It is a standard markup language for creating Web pages. It describes the structure of a Web page and consists of a series of elements. HTML elements tell the browser how to display the content. HTML elements are represented by tags. Each HTML tag describes different content.

History

HTML was first created by Tim Berners-Lee, Robert Cailliau, and others starting in 1989. It stands for Hyper Text Markup Language. Hypertext means that the document contains links that allow the reader to jump to other places in the document or to another document altogether. The latest version of HTML is HTML5, which was published in October 2014 by the W3C. This version contains new and efficient ways of handling elements such as video and audio files.

HTML was first proposed by Tim Berners-Lee in 1989 and later published in 1991. HTML 2.0 was published in 1995, HTML 3.2 in 1997, HTML 4.01 in 1999, XHTML 1.0 in 2000, and HTML5 in 2014.

Development

Development of HTML was driven by the need to define the structure of documents and ensure the right tags were used to define the right elements. The first version of HTML was written by Tim Berners-Lee in 1991. Since then, the HTML specification has gone through many changes and updates. The latest version of HTML is HTML5, which was published in October 2014 by the W3C.

HTML5 is the latest evolution of the standard that defines HTML. The term represents two different concepts. It is a new version of the language HTML, with new elements, attributes, and behaviors, and a larger set of technologies that allows the building of more diverse and powerful Web sites and applications. This set is sometimes called HTML5 & friends and often shortened to just HTML5.

Usage

HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page and consists of a series of elements. These elements tell the browser how to display the content. HTML elements are represented by tags. Each HTML tag describes different content.

HTML is used to create electronic documents (called pages) that are displayed on the World Wide Web. Each page contains a series of connections to other pages called hyperlinks. Every web page you see on the Internet is written using one version of HTML code or another.

Usage of HTML is not limited to just web pages. HTML is also used to create email templates, and to create a structure for content in a word processing tool.
